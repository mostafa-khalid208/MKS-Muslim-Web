## 📖 Overview | نبذة عامة

**MKS Muslim Web** is a modern, responsive web application designed to bring the premium experience of the MKS Muslim mobile app to the web. It uses Progressive Web App (PWA) technology, allowing you to install it directly on your desktop or mobile device for a native-like experience.

**تطبيق المسلم للويب (MKS Muslim Web)** هو تطبيق ويب حديث ومتجاوب صُمم لينقل التجربة الفاخرة لتطبيق الهاتف المحمول إلى الويب. يستخدم التطبيق تقنية تطبيقات الويب التقدمية (PWA)، مما يتيح لك تثبيته مباشرة على جهاز الكمبيوتر أو الهاتف للحصول على تجربة تشبه التطبيقات الأصلية.

## ⭐ Key Features | المميزات الرئيسية

### 🕋 1. Prayer Times & Adhan | مواقيت الصلاة والأذان
*   **Precision Accuracy**: Timings calculated via Aladhan API based on your location.
    *   *دقة عالية:* حساب المواقيت عبر واجهة Aladhan بناءً على موقعك.
*   **Adhan Audio**: Beautiful Adhan played at prayer times.
    *   *صوت الأذان:* تشغيل صوت الأذان العذب في أوقات الصلاة.
*   **Remaining Time**: Real-time countdown to the next prayer.
    *   *الوقت المتبقي:* عد تنازلي حي للصلاة القادمة.

### 📖 2. The Noble Quran | القرآن الكريم
*   **Complete Mushaf**: High-quality text with clear typography.
    *   *مصحف كامل:* نصوص عالية الجودة مع خط واضح.
*   **Audio Recitations**: Integrated audio player for every verse and Surah.
    *   *تلاوات صوتية:* مشغل صوتي مدمج لكل آية وسورة.
*   **Bookmarks & Last Read**: 
    *   Save your progress automatically or bookmark specific verses.
    *   *العلامات وآخر قراءة:* حفظ تقدمك تلقائياً أو وضع علامات على آيات محددة.

### 🧭 3. Qibla Compass | اتجاه القبلة
*   **Visual Guidance**: Smooth, animated compass working on mobile devices.
    *   *بوصلة بصرية:* إبرة متحركة سلسة تعمل على الهواتف المحمولة.
*   **Status Indicators**: Clear feedback on GPS accuracy.
    *   *مؤشرات الحالة:* تعليقات واضحة حول دقة نظام تحديد المواقع.

### 📚 4. Authentic Knowledge | العلم الشرعي والأذكار
*   **Hadith Library**: Selections from **Sahih Al-Bukhari** & **Sahih Muslim**.
    *   *مكتبة الحديث:* مختارات من صحيح البخاري وصحيح مسلم.
*   **Azkar (Adhkar)**: Complete Morning, Evening, and Sleep remembrances.
    *   *الأذكار:* أذكار الصباح والمساء والنوم كاملة.
*   **Supplications (Duas)**: Categorized Duas for various needs.
    *   *الأدعية:* أدعية مبوبة لاحتياجات متنوعة.
*   **Sunnah Prayers**: Guides for Duha, Qiyam, Istikhara, and more.
    *   *سنن الصلوات:* أدلة لصلاة الضحى، القيـام، الاستخارة، وغيرها.

### ⚙️ 5. Productivity Tools | أدوات الإنتاجية
*   **Tasks System**: Integrated To-Do list to manage your daily worship goals.
    *   *نظام المهام:* قائمة مهام مدمجة لإدارة أهدافك اليومية.
*   **Digital Tasbih**: Interactive counter with vibration feedback (mobile).
    *   *السبحة الإلكترونية:* عداد تفاعلي مع اهتزاز (على الجوال).

## 💻 Installation (PWA) | التثبيت

**Desktop (Windows/Mac)**:
1. Open the website in Chrome or Edge.
2. Look for the "Install" icon in the address bar (top right).
3. Click **"Install MKS Muslim"**.
4. The app will open in its own window and appear on your desktop/start menu.

**Mobile (Android/iOS)**:
1. Open in Chrome (Android) or Safari (iOS).
2. Tap "Add to Home Screen".
3. Enjoy the full-screen experience.

## 🌍 Tech Stack | التقنيات المستخدمة

*   **Frontend**: HTML5, CSS3 (Variables, Flexbox/Grid), JavaScript (ES6+).
*   **PWA**: Service Workers, Manifest V2.
*   **APIs**: Aladhan API (Prayer Times, Hijri Date), Quran Cloud API.
*   **Storage**: LocalStorage for persistent user data.
